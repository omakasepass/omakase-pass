The site already references them — no code changes needed once they're here.

| Filename     | What it is                    | Notes / size |
|--------------|-------------------------------|--------------|

Until these exist, the site shows tasteful dark placeholder blocks in their place —
so it still looks intentional if you deploy before the photos are ready.

**Licensing:** use images you own or have licensed for commercial use
(e.g. paid stock from Adobe Stock / Getty, or a chef's own photos with permission).
Avoid grabbing random photos from Google Images.
